🚀 Install the Real Version

The actual WiFi Radar runs on your Mac and scans real WiFi networks:

# Clone or download the Python files
python3 wifi_radar_ui.py

# Or use the launcher
bash launch_radar.sh

How It Works

  • Scans WiFi networks every 2 seconds using macOS airport utility
  • Builds a baseline of signal strengths over 40 seconds
  • Detects motion when multiple APs show unusual signal fluctuations
  • Visual radar display with real-time updates

Requirements

  • macOS (uses airport command)
  • Python 3.7+
  • Tkinter (built into Python)
